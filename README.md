# Portfolio

Personal portfolio website for Sashank Varma Rudraraju, Machine Learning Engineer.

## Tech Stack

- Next.js 14
- Tailwind CSS
- Framer Motion
- Recharts
- TypeScript

## Features

- Responsive design with dark/light mode
- Animated hero section with profile photo
- Experience timeline with metrics
- Projects showcase with interactive demos
- Skills visualization with radar chart
- Blog section with markdown content

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
```

## Deployment

Deployed to GitHub Pages via GitHub Actions.

Live site: https://sashankvarma2000.github.io/portfolio
